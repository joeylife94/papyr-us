**이 프로젝트는 Docker 환경에서 개발하는 것을 권장합니다.**

### Docker 환경 실행
```bash
# 서버 시작
docker-compose up --build

# 백그라운드 실행
docker-compose up -d --build

# 서버 재시작 (API 변경사항 반영 시)
docker-compose restart
```
