## 📋 문서 관리 가이드

### 문서 업데이트 시 체크리스트

- [ ] 이 인덱스 파일 업데이트
- [ ] 날짜 및 버전 정보 갱신
- [ ] 관련 문서 간 링크 확인
- [ ] 변경 사항 히스토리 기록

### 문서 작성 규칙

1. 모든 문서는 마크다운 형식으로 작성
2. 제목은 명확하고 간결하게
3. 코드 예시는 언어 명시 (`typescript, `bash 등)
4. 스크린샷은 `docs/assets/` 디렉토리에 저장
5. 외부 링크는 절대 경로 사용
